<h1 align="left">Bike Sales Dashboard</h1>
This repository contains an Excel-based data analysis project focused on bike purchase behavior across different demographic segments. 
The workflow includes data cleaning, transformation, and visualization to derive actionable insights.

<h1 align="left">Section description</h1>
- Cleaned and structured raw dataset for analysis
- Pivot tables for aggregating and segmenting data
- Visualizations (charts) to highlight trends and patterns
- Interactive dashboard with slicers for dynamic filtering

<h1 align="left">Final result</h1>
The final dashboard enables exploration of how bike purchases vary by gender, education level, and region, making it easy to identify which customer segments have the highest purchase rates.

<h1 align="left">Tools used</h1>
Microsoft Excel (data cleaning, pivot tables, charts, dashboard design)
